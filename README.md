# Quiz App

## Setup Instructions

**Prerequisites:**
- Node.js (v18 or above recommended)

**Steps:**
1. Clone the repository or download the source code.
2. Install dependencies:
   ```bash
   npm install
   ```
4. Run the app locally:
   ```bash
   npm run dev
   ```
5. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack Used
- React (with functional components and hooks)
- Vite (for fast development and build)
- Tailwind CSS (utility-first styling)
- Framer Motion (animations and transitions)

## Key Features Implemented
- Landing page with animated arrival and start button
- Quiz flow with animated transitions between questions
- Cinematic blur/scale arrival for all major components
- Progress bar with dynamic thickness (thin base, thick progress)
- Responsive design for desktop and mobile
- Result page with smooth container expansion animation
- Custom arrow buttons and option buttons with gradients and hover effects

## Assumptions Made
- The quiz questions are static and provided in the code (no API integration)
- User can restart the quiz after finishing
- No authentication or user tracking
- Design and animation choices are based on provided references and user feedback

## Time Spent on the Assignment
+- Initial setup and scaffolding: ~30 minutes

+- Animation logic and Framer Motion integration: ~1 hour
+- Iterative feedback and refinements: ~1.5 hours
+- Bug fixes and polish: ~1 hour
